A Time Management egyszerű időnyilvántartó rendszer telepítését az alábbiakban részletezem:

1. Kicsomagolás után egy fejlesztői környezet választása után húzzuk be a projektet utána a fejlesztői környezet terminálját nyissuk meg.
2. A terminálban a projekt könyvtárjában, ha nem vagyunk benne, akkor a "cd '.\Time Management\'" paranccsal beleléphetünk.
3. A terminálba egy "npm run build" vagy "yarn build" parancs segítségével feltelepíti a csomagokat. 
4. A terminálba helyi szervert nyitunk az "npm run dev" vagy "yarn run dev" paranccsal.
5. Végül a terminálban feltűntetett Local host linkre kattintva a böngészőben megnyílik a program.
